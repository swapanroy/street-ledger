Let's connect an AI to read your photos

This is a one-time, 2-minute setup. You'll get a free code from Google, paste it below, and you're done. Nobody but Google ever sees this code — not us.

  1. Click the green button below — it opens Google's site in a new tab.
  2. Sign in with any Google account (the same one you use for Gmail is fine).
  3. Click "Create API key", then copy the code it gives you.
  4. Come back to this tab, paste the code in the box below, and click Connect.
